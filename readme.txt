◆このソフトについて
  Ancia(アンシア)は、軽い動作を目指し設計され、多彩な操作性を持つ国産ブラウザです。

  次の特徴があります。

  ・HiDPI表示に対応
  ・タブのマルチスレッド/マルチプロセス動作
  ・タブグループ機能
  ・互換表示
  ・他ブラウザのブックマークがそのまま使用可(IE,Chrome,Firefox,Operaをサポート)
  ・固まったタブを強制終了できるタブマネージャ機能
  ・ドラッグ＆ドロップでの様々な操作(ドロップアクション)
  ・ユーザ定義スクリプトが実行可能なURLアクション
  ・ボタン式、または、カーソル式のマウスジェスチャ
  ・IEプラグインの使用
  ・検索候補表示・検索キーワード自動ハイライト
  ・インクリメンタル検索
  ・オートテキストリンク/イメージリサイズ
  ・多段タブ
  ・ユーザ定義スクリプトの実行
  ・ページ単位で設定できるユーザスタイルシート
  ・レジストリ未使用(例外あり※)
  ・ライブサムネイル,AeroなどWindows7対応済み
  ・シングルキーショートカット
  ・タブの別ウインドウへの切り離し・結合
  ・スーパードラッグ＆ドロップ
  ・検索バー、アドレスバーでのホイールクリック貼り付け検索や移動

◆インストール方法・バージョンアップ方法
  インストールの必要はありません。解凍したファイルに含まれるAncia.exeがアプリケー
  ションです。
  Program Files配下に置くことは、設定ファイルがWindowsにより別の場所に保存されるため、非推奨です。

◆アンインストール方法
  ダウンロードしたファイルを全て削除してください。
  アプリケーションを既定のブラウザに設定したときは、解除してから削除することを
  推奨します。

◆レジストリについて※
  IEを最新の状態で使用するためのレジストリは利用しています。このレジストリには、
  アプリケーションの動作情報は含まれていません。オプション設定に使用している
  レジストリが書いてあります。

◆表示トラブル
  Kasperskyインストール状態で、google関連サイト(https://www.google.co.jp/など)のページが開けない場合 ‡
  Kaspersky Internet Security 2012がインストールされているとき、下記のようなGoogle関連サイト
    ・https://www.google.co.jp/
    ・http://www.google.co.jp/
    ・その他、Gmail関係サイトなど
  開けない場合、Kaspersky Internet Security 2012の設定変更を

  ============================================
  設定>ネットワーク>プロキシサーバーとネットワークトラフィック制御の設定>
  ネットワーク>暗号化された接続>暗号化された接続をスキャンで、
  ｢暗号化された接続をスキャン｣を｢有効｣→｢無効｣に変更
  ============================================

  することで表示される報告をいただいています(g13131gさんありがとうございます)。
